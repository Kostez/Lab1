# Lab1
How to build:

  make
  
How to clearn:

  make clearn
  
How to run in modes:

    std:    ./bin/lab --mode=std
    child:  ./bin/lab --mode=child
    posix:  ./bin/lab --mode=posix --amount=N
    kill:   ./bin/lab --mode=kill --signal=SIGNO --pid=VALIDKILLPI
    pipe:   ./bin/lab --mode=pipe
